How to Use

1. Set up the table

Add players, starting stacks and (optionally) a photo for each. Set the blinds, then tap Shuffle Up & Deal.

2. Every hand

Tap the deck to shuffle and deal hole cards. Tap a name in the gold Dealer box to move the button before the deal (it locks once a hand starts).

3. Passing the phone

Before each turn you'll see "Pass the phone". Hand it over. That player taps to reveal, sees only their own cards, chooses an action, then taps End Turn, which hides everything again before the next pass.

4. Streets & side pots

Flop, turn and river deal automatically once betting closes. Hold the phone up so the table can see. At showdown, remaining hands are shown together and you tap the winner(s); side pots from all-ins split correctly on their own.
